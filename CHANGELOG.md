All notable changes to TransitKit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15
### Added
- Initial release of TransitKit
- Core transit modeling with Mandel & Agol 2002
- Support for TESS, Kepler, K2, CHEOPS data
- MCMC, nested sampling, and least-squares fitting
- Interactive web interface (Streamlit)
- Desktop application (PyQt6)
- Jupyter notebook integration
- Publication-quality figure generation
- Multi-planet system analysis
- TTV analysis tools
- Transmission spectroscopy pipeline
- Plugin system for extensibility
- Comprehensive documentation
- Example notebooks and tutorials
