Introduction
------------

TransitKit is a toolkit for analyzing exoplanet transit light curves.

Installation
------------

.. code-block:: bash

   pip install transitkit

Quick Start
-----------

.. code-block:: python

   import transitkit as tk
   print(f"Version: {tk.__version__}")
