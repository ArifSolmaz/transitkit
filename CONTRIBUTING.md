# Contributing to TransitKit

Thank you for considering contributing to TransitKit!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork
3. **Create a branch** for your feature/fix
4. **Make changes** and add tests
5. **Run tests** to ensure nothing breaks
6. **Submit a Pull Request**

## Development Setup

```bash
# Clone your fork
git clone https://github.com/yourusername/transitkit.git
cd transitkit

# Install in development mode
pip install -e .[dev]

# Run tests
pytest tests/
