Validation Module

Transit validation and quality checks.

Validation tools for transit analysis results.

transitkit.validation.assess_multiple_event_statistic(time, flux, period, t0, duration)[source]

Calculate Multiple Event Statistic (MES) for transit detection. Similar to Kepler pipeline.

transitkit.validation.calculate_detection_significance(bls_result, n_shuffles=1000)[source]

Calculate detection significance via data shuffling.

transitkit.validation.compare_with_known_ephemeris(params, known_params, tolerance=0.01)[source]

Compare detected parameters with known ephemeris.

transitkit.validation.perform_injection_recovery_test(time, injection_params, n_trials=100, noise_level=0.001, seed=42)[source]

Perform injection-recovery test to assess detection efficiency.

transitkit.validation.validate_against_secondary_eclipse(time, flux, period, t0, duration)[source]

Check for secondary eclipse to validate planet detection. Secondary eclipse at phase 0.5 if eccentric.

transitkit.validation.validate_transit_parameters(params, time, flux)[source]

Validate transit parameters against physical limits.
